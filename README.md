# Python ToDo App

Python学習のアウトプットとして作成した、CUI（ターミナル）で動作するシンプルなToDoアプリです。

## 概要
ターミナル上でコマンドを入力し、タスクの追加・一覧表示・完了（削除）を行うToDoアプリです。  
Pythonの基本文法や処理の流れを理解することを目的に作成しました。

## 操作方法
以下のコマンドを入力して操作します。

- `add`：タスクを追加する
- `show`：現在のタスク一覧を表示する
- `done`：番号を指定してタスクを完了（削除）する
- `exit`：アプリを終了する

## 使用技術
- Python 3
